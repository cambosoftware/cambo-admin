## Quick Installation

```bash
# Install via Composer
composer require cambosoftware/cambo-admin

# Run the interactive installer
php artisan cambo:install
```

## Main Features

- **Complete Authentication** - Login, register, 2FA, sessions
- **Roles & Permissions** - Flexible RBAC system
- **Notifications** - Real-time with database storage
- **Activity Log** - Complete action tracking
- **Dashboard** - Customizable widgets and statistics
- **File Manager** - Upload and media management
- **Settings** - Dynamic configuration
- **Import/Export** - CSV, Excel, PDF
- **Internationalization** - Multi-language with RTL support
- **Themes** - Dark mode and custom themes
- **CLI** - Powerful code generators

## Why CamboAdmin?

CamboAdmin is designed for Laravel developers who want to create professional admin panels quickly, without sacrificing code quality or maintainability.

### Modern Architecture

- **Laravel 12** with best practices
- **Vue 3 Composition API** for reactive components
- **Inertia.js v2** for SPA experience without API
- **TypeScript ready** for more type safety

### Maximum Productivity

- **134+ Vue components** ready to use
- **CLI generators** to automate repetitive code
- **Complete CRUD templates**
- **Comprehensive documentation**
