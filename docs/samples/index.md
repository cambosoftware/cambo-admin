# Sample Pages

Ready-to-use page templates built entirely with CamboAdmin components. Copy and paste directly into your project.

## Available Samples

### Authentication

- [Login Page](./login.md) - Modern login form with validation
- [Register Page](./register.md) - User registration with password strength

### Dashboard

- [Admin Dashboard](./dashboard.md) - Complete dashboard with stats, charts, and tables

### CRUD Pages

- [User List](./users/index.md) - Data table with search, filters, and bulk actions
- [User Create](./users/create.md) - Create form with validation

### Other Pages

- [Settings](./settings.md) - Application settings with tabs

## How to Use

1. Browse the sample pages to find what you need
2. Click on a sample to view the full code
3. Copy the Vue component code
4. Paste into your project and customize

All samples use:
- Vue 3 Composition API (`<script setup>`)
- CamboAdmin components
- Tailwind CSS utility classes
- Inertia.js patterns where applicable
